toolstip will help you to explain the event or content in the shot form popup if you hover on it

<button type="button" class="btn btn-info" data-bs-toggle="tooltip" title="...content">

you can see it does not work, we need to add a js implement so that it work

let tooltipTriggerList = [].slice.call(document.querySelectorAll('[data-bs-toggle="tooltip"]'));
let tooltipList = tooltipTriggerList.map(function (tooltipTriggerList){
    return new bootstrap.Tooltip(tooltipTriggerList);
})

we can placement the position of tooltip with class data-bs-placement ="right|left|top|bottom"

popover works like tooltip but it contain more information

to create a popover we need to add attribute data-bs-toggle="popover" with titel="..." data-bs-content="..." into a button or a tag

click to see the effect of popover, note that we need to implement js code like tooltip so that it works

let popoverTriggerList = [].slice.call(document.querySelectorAll('[data-bs-toggle="popover"]'));
let popoverList = popoverTriggerList.map(funtion (popoverTriggerList){
    return new bootstrap.Popover(popoverTriggerList);
})

we can also place the position of popover with data-bs-placement like tooltip

we can also hover to toggle popover with data-bs-trigger="hover"

toast provides a alert box for a couple seconds when something happpens

when do we use toast?

  • to show a alert of an action like success, fail
  • not to interupt focus of user in term of user experience
  • system information

use a container with class toast with 2 children container toast-header,toast-body to create a toast

use class show to display toast as default and if you want to close, add a button and add data-bs-dismiss="toast"

tip: use class me-auto to modify position of btn-close

how to create a offcanvas?

<div class="offcanvas offcanvas-start">
    <div class="offcanvas-header" id="idName">
        <strong class="offcanvas-title"></strong>
        <button class="btn-dismiss" type="button" data-bs-dismiss="offcanvas"></div>
    <div class="offcanvas-body">
        <p> content</p>
</div>

<button class="btn btn-primary" type="button" data-bs-target="idName" da-bs-toggle="offcanvas">click to see</button>

you can also use class="offcanvas-end|top|bottom| to modify position of offcanvas.

using the class="offcanvas-sm|md|lg|xl|xxl" to determine how offcanva responsive.

tip: using class='text-bg-dark' in father container and button class="btn-close btn-close-white" to create a dark offcanvas.
