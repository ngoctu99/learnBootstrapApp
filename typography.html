Bootstrap 5 default settings

Bootstrap 5 uses a default font-size of 1rem = 16px, and it line-height ist 1.5

By the way the tag "p" has a default margin-top 0 and margin-bottom 1rem = 16px

The following classes are used to modify width of the text

Bootstrap uses the following for text-color:

  • .text-primary text
  • .text-secondary text
  • .text-success text
  • text-danger text
  • .text-warning text-warning
  • .text-info text-info
  • .text-light text-light
  • .text-dark text-dark
  • .text-body text-body
  • .text-mute text-muted
  • .text-black-50 text-black-50
  • .text-white-50 text-white-50

Class .display will be used to stand out more than the normal headings with larger fontsize and lighter fontweight
